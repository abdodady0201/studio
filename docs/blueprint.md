# **App Name**: Holy Quran - Recitation and Interpretation Offline

## Core Features:

- Quran Display & Navigation: View the complete 114 Surahs with a clear Uthmani font, adjustable font size, and toggle between night and day modes. Navigate by Surah, Juz, Hizb, or Rub.
- Offline Audio Recitation: Listen to full Surah recitations from various Qaris (e.g., Sudais, Muaiqly, Ghamdi). Download Surahs for offline playback with background listening and media notification controls.
- Advanced Text Search: Perform word-based searches across the Quran, displaying results with highlighted keywords and direct navigation to the corresponding verse.
- Interpretation (Tafsir): Access Tafsir from Ibn Katheer and Al-Saadi. Interpretations are downloaded on-demand to optimize application size.
- Bookmarks & Progress Tracking: Save favorite verses, add personal notes, automatically track the last reading position, and monitor 'Khatmah Quran' progress with completion percentages.
- Adhkar & Tasbeeh Counter: Display Morning and Evening Adhkar, post-prayer supplications, and include a digital Tasbeeh counter.
- Accurate Prayer Times: Automatically detect location, calculate precise prayer times based on the user's country, and provide optional Adhan alerts.

## Style Guidelines:

- Primary color: Deep, rich emerald green (#196643) to convey depth and spirituality in a dark theme.
- Background color: A very dark, desaturated greenish-grey (#161C19) to create a serene reading environment, especially for night mode.
- Accent color: A vibrant golden yellow (#E5B11A) used for highlights, interactive elements, and to symbolize richness and value.
- Headline font: 'PT Sans' (humanist sans-serif) for clean and modern headings and UI elements. Body font: 'Literata' (transitional serif) for an elegant and readable presentation of Quranic verses and longer texts.
- Use professionally designed icons that are clear and subtly outlined, complementing the rich and serene color palette.
- Implement a 'Bottom Navigation' bar for intuitive access to core sections. A welcoming splash screen will display the phrase 'اقرأ وارتقِ' ('Read and Ascend').
- Incorporate subtle, smooth transitions and micro-animations to enhance user interaction without distraction, maintaining a serene user experience.
